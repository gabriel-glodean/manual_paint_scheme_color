# ModelPaintingSchemeHelper

## Backend (FastAPI)
- Go to `backend/app/`
- Install dependencies: `pip install -r requirements.txt`
- Run: `uvicorn main:app --reload`

## Frontend (React)
- Go to `frontend/`
- Install dependencies: `npm install`
- Run: `npm start`

## Usage
- Open http://localhost:3000 for the React app
- The React app will call the FastAPI backend (default: http://localhost:8000)
